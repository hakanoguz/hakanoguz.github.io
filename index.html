Python

I teach a graduate level course of Python Programming at KSU. I have also written TOPP-The Optimal Path Program in python. This tool accepts a DEM (Digital Elevation Model) image and then outputs the optimal path in raster and vector (shapefile) format.

Web Programming

I teach another graduate level course of Web Programming with JavaScript at the same university. I teach HTML, CSS, JavaScript, and DJango in this course.

iOS App Development

I also teach iOS App Development course. This course covers swift programming language and XCode IDE.

VB .NET

I have taught Computer Programming with VB .NET course for more than 10 years. I have developed a number of tools using VB .NET such as LST Calculator, which retrieves land surface temperature from Landsat 5, 7, 8 and 9 imageries. I have also developed HOPP-Hakan's Optimal Path tool using VB .NET.
